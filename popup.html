Formative Marks Helper

This extension automates filling formative marks on the Banglar Shiksha portal.

How to use:

  1. Install the extension.
  2. Open the marks entry page.
  3. Use the banner at the top of the page to input marks.

Disclaimer: This tool is provided as-is. Use it responsibly. Ensure compliance with institutional policies before using.

Limitations: Works only on https://school.banglarshiksha.gov.in/sms/exam/formative_marks_edit.
